# @workflow/utils

Utility functions for Workflow DevKit.

## Installation

```bash
npm install @workflow/utils
```

## Features

- **`parseDurationToDate()`** - Parses duration strings, numbers, or Date objects to a Date
- **`withResolvers()`** - Polyfill for `Promise.withResolvers()`
- **`once()`** - Creates a lazily-evaluated, memoized function
- **`PromiseWithResolvers`** - Type interface for promise resolvers
